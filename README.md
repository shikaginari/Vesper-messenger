Vesper Messenger

Vesper — это современное десктопное приложение для обмена сообщениями, ориентированное на групповое общение. Проект реализован на стеке Java с использованием JavaFX для графического интерфейса и PostgreSQL для надежного хранения данных.

🚀 Основные возможности

Управление группами: Создание чатов, динамическое добавление и исключение участников, смена названия группы.
Интерактивная палитра эмодзи: Контекстный поиск по русскоязычным тегам (например, «котик», «кубок»).
Настраиваемое поведение (автозакрытие панели после выбора).
Надежное хранение данных: Использование PostgreSQL с поддержкой каскадного удаления для обеспечения целостности истории переписки.
Адаптивный UI: Современный дизайн на JavaFX с использованием Glass-pane эффектов и удобной навигацией.
Стабильность: Покрытие критических сценариев интеграционными и модульными тестами.

🛠 Технологический стек

Язык: Java 21+
UI: JavaFX
БД: PostgreSQL
Сетевой уровень: Netty
Логирование: SLF4J + Logback
Тестирование: JUnit 5, Mockito
